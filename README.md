# Railway Track Crack Detection System

## Overview

The Railway Track Crack Detection is a detection as well as  an alert system . This is an embedded systems project designed to improve railway safety through the early detection of track faults and discontinuities. Railway track failures can lead to severe accidents, operational disruptions, and significant economic losses. This project addresses this challenge by continuously monitoring track conditions and generating alerts whenever a crack or break in the railway track is detected.

The system was designed and validated using Tinkercad simulation and demonstrates the application of Arduino-based embedded systems in transportation safety and infrastructure monitoring.

---

## Problem Statement

Railway track cracks and structural failures are among the leading causes of railway accidents worldwide. Manual inspection methods are time-consuming, costly, and may fail to detect defects in real time.

The objective of this project is to develop an automated monitoring system capable of detecting track discontinuities and immediately alerting railway authorities to potential hazards.

## Working Principle

The system continuously monitors the condition of the railway track through sensors placed at predefined locations.

### Normal Condition

* Track continuity is maintained.
* Sensors indicate a healthy track condition.
* No alert is generated.
* Green indicator remains active.

### Crack Detected Condition

* Track continuity is interrupted.
* Sensor state changes immediately.
* Arduino processes the fault signal.
* Warning LED and buzzer are activated.
* Maintenance personnel can be notified for inspection and corrective action.

This enables rapid identification of potential hazards before they develop into serious safety incidents.
